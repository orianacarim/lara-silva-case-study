# lara-silva-case-study

Landing editorial / case study para presentar la identidad visual y verbal de Lara Silva, psicóloga clínica con enfoque existencial.

El sitio está construido como una página estática en HTML, CSS y JavaScript, lista para publicar en GitHub Pages.

## Abrir localmente

Abrir `index.html` en el navegador o levantar un servidor local:

```bash
python3 -m http.server 8080
```

Luego entrar a:

```text
http://localhost:8080
```

## Publicar En GitHub Pages

1. Crear un repositorio llamado `lara-silva-case-study`.
2. Subir el contenido de esta carpeta.
3. En GitHub, ir a `Settings > Pages`.
4. Elegir `Deploy from a branch`.
5. Seleccionar la rama `main` y la carpeta `/root`.

## Fuentes

La hoja usa Boska y Satoshi desde Fontshare. Si se quiere dejar todo 100% offline, conviene reemplazar ese import por archivos locales de fuentes en una carpeta `assets/fonts`.
